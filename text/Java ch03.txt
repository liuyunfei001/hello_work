二、选择结构
  1.什么是选择结构
    选择结构是根据条件判断之后再做处理
  
  2.选择结构的语句:if语句,switch语句

三、if语句
  1.if语句的语法:
    if(条件){
      条件成立时执行的代码块
    }else{
      条件不成立时执行的代码块
    }
    
   注意:a.条件一般为关系表达式，逻辑表达或返回结果为bolean类型的任何表达式
        b.else块可有可无
        c.当满足条件时，执行的代码块只有一行时可省大括号.

  2.嵌套if语句
    所谓嵌套if就是在if语中在放if语句:
    语法:
    if(条件){
       if(条件){}else{}
    }else{
       if(条件){}else{}
    }

  3.多重if语句
   if(条件){
      代码块
   }else if(条件){
      代码块
   }else if(条件){
   代码块
   }else{....}

  
四、逻辑运算符
1.作用：连接多个关系表达式  
  例如:表达式1 &&  表达式2  
2.运算符:
  &&  左右两边同为真时为真
  ||  左右两边有一边为真即为真
  !   取反(非真非假)
