一、变量
 1.什么是变量:在程序运行过程中，暂时保存信息(数据)的量称为变量.
 2.使用变量
	2.1.定义变量
		//数据类型 变量名;
		//int math;
	2.2.变量赋值
		//变量名=值;
		//math=5;
	2.3.获取变量的值
	       // 变量名

	2.4.定义变量并初始化值
		int math=5;

3.变量名的命名规范
  变量名是由任意数字，字母，下划线，$构成，且不能以数字开头，不能使用java关键字
  注意:变量名应做到见名知义


二、数据类型
 作用:区分变量存不同的数据
 数据类型分类:
  a.数值型:
   整型:byte字节,short短整型,int整型,long长整型
   非整型:float单精度浮点，double双精度浮点
  b.非数值型:
   char字符型(存储单个字符，字符需要用单引号表示:如:'a')
   boolean(布尔型):存真(true)假(false)值
   String字符串型(存储多个字符，字符串需要用双引号表示:如:"abc")
  注意:除String以外的数据类型就是java的八大基本类型.

三、运算符表达式
 1.运算符分类:
   赋值运算符:=
   算术运算符:+,-,*,/,%(取余，取模),++自加,--自减
   关系运算符:<,<=,>=,>,==(等于),!=(不等于)
   逻辑运算符:||(或),&&(且),!(非)
 
 2.表达式
  表达式=操作数+运算符
  注意:用什么运算符连接的式子即称为什么表达式
  
 3.运算符优先级
 ()>算术运算符>关系运算符>逻辑运算符
 
四、java使用Scanner类实现输入的步骤
  1.导入Scanner类
    import java.util.Scanner;  或者 import java.util.*;
  2.创建Scanner类的对象
    //Scanner 对象名=new Scanner(System.in);  
    Scanner input=new Scanner(System.in);  
  3.接收用户输入信息
     int 变量名=input.nextInt(); //接收输入整数
     double 变量名=input.nextDouble(); //接收输入小数
     String 变量名=input.next();  //接收输入字符串

五、类型转换
 1.基本类型由低到高的顺序
   char->short->int->long->float->double
 2.隐式转换:
   类型由低到高自动转换
 3.强制转换
   类型由高到低强制转换
   语法:(目标类型)变量;
